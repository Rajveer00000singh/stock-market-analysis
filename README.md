# Indian Stock Market Analysis

## Project Overview
Analysis of 5 years of real NSE stock data (2020-2025) for 9 major 
Indian companies across 5 sectors using Python and Power BI.

## Tools Used
- **Python** — Data fetching and analysis
- **yfinance API** — Real NSE stock data
- **Pandas, Matplotlib** — Data processing and visualization
- **Power BI** — Interactive dashboard
- **Excel** — Data storage and export

## Companies Analyzed
| Company | Sector |
|---------|--------|
| TCS, Infosys, Wipro | IT |
| HDFC Bank, ICICI Bank | Banking |
| Reliance, ONGC | Energy |
| Maruti | Auto |
| HUL | FMCG |

## Key Findings
1. **Infosys** gave highest 5-year return — **188%**
2. **ONGC** was highest risk-highest return stock — **160% return**
3. **HUL** was most stable stock — lowest volatility
4. **IT Sector** dominated long-term returns
5. High Risk = High Return pattern proven across all sectors

## Dashboard
Interactive Power BI dashboard with:
- 5 Year Total Returns comparison
- Sector wise average returns
- Risk vs Return scatter plot
- Stock price trend (2020-2025)
- Sector filter slicer

## Dataset
- Source: Yahoo Finance (yfinance API)
- 9 major NSE listed companies
- 5 years of daily data (2020-2025)
- ~1238 trading days per stock
